# IELTS ContextVocab (V1.0.0)

这是一个帮你在语境中（阅读短文）记忆雅思单词的小工具。

写这个工具的初衷是觉得光背单词表很枯燥且容易忘。它能把你每天要背的单词表投射到一篇短文中，自动标红生词。你在读文章的时候，顺手把认识的词点掉，不认识的词右键复制去查，“在语境中消灭生词”。

---

## 功能

* 只高亮生词：导入当天的词表和短文，软件会自动把文中的生词加粗标红。
* 首现高亮：为了不影响阅读体验，同一个生词只在第一次出现时标红，后面再出现时保持原样。
* 点一下就“认识”：
    * 左键点击红色的词 -> 变灰（标记为已掌握）。
    * 右键点击任意单词 -> 自动复制到剪贴板（方便查词典）。
* 自动查漏：如果短文没包含词表里的所有词，软件会自动把漏掉的词列在文章最后，不用担心漏背。
* 自动存进度：随时关闭软件，下次打开会自动恢复你背到的位置（红/灰状态）。
* 生词导出：背完一章后，可以一键把还没记住的词导出成 txt，方便导入 Anki 或者重点复习。

---

## 用法

这个工具只用了 Python 自带的库（Tkinter），没有复杂的依赖。

如果你电脑上有 `uv`，直接在项目根目录运行：
```bash
uv run app.py
```
有`Python`也行

```bash
python app.py
```
