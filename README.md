# IELTS ContextVocab (V1.0.0)

IELTS ContextVocab 是一款专为雅思备考者设计的语境生词过滤工具。它改变了传统的死记硬背模式，通过将每日词表嵌入到精心编写的短文中，帮助用户在阅读中识别生词、标记熟词，并实现备考进度的自动化管理。

## 🌟 核心特性

- 语境记忆法：自动关联词表与短文，在实际语境中复习单词。
- 生词动态过滤：
    - 左键点击：切换单词状态（红色加粗为“生词”，灰色普通为“已掌握”）。
    - 首现高亮：文中多次出现同一单词时，仅高亮第一次，保持阅读流畅性。
- 缺失词补全：自动检查并罗列短文中未涵盖的词表单词，确保复习无死角。
- 智能导航：支持“上一天”、“下一天”快捷切换，自动恢复上次学习进度。
- 高效查词辅助：右键点击文中任意单词即可自动复制到剪贴板。
- 数据持久化：自动保存每日掌握进度，支持导出生词本用于二次复习。
